Welcome to INGES

International Global Educational System

Objective

This project objective is to create a global educational platform, in order to provide basics highly relevant knowledge to any person anywhere in this world, trying to reach enough level to make significant change in it's users' life experience.

How to do it

Building studying flows based on best known learning techniques as possible as:

  • Languages teaching
  • Clear scopes and objectives, straight to the point
  • Spaced repetition and task management
  • Effective Habits lessons, character and principles first
  • Smart community profiles

Technologies around on this system

This system is based on a frontend application hosted on the cloud, an API connection manager hosted on another cloud and a Business API hosted that provides the database artifacts, users records and etc.

Warning notes

Eventually the business server (which record your account data) will be off-line because it's currently hosted on a machine which isn't everytime online. As soon as we get enough money, we will provide a dedicated 24h server.
