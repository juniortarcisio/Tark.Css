Welcome to INGES

International Global Educational System

Objective

This project objective is to create an international global educational platform, in order to provide the basic most highly relevant knowledge to any person anywhere in this world, trying to reach enough level to make significant change on it's users' lives experiences.

The secondary objective of this project is being a personal creativity portifolio, exercising my not so great languages skills and my professional developer techniques as well keeping myself excited and updated in this constant moving world.

How to reach this objective?

Building studying flows based on best known learning techniques as possible as:

  • Languages teaching and multiculturalism encouraging (drastically widening your possibilities)
  • Effective habits lessons, based on character and valuable principles first
  • Task management and Spaced repetition
  • Clear scopes and objectives, straight to the point
  • Smart community profiles

How to use this system?

1. Set your languages

Select one of yours mastered languages and your learning language on topbar

2. Browse using the menu

Use the side menu to choose the topic of your interest

Technologies around on this system

This system is based on a frontend application hosted on the cloud, an API connection manager hosted on another cloud and a Business API hosted that provides the database artifacts, users records and etc.

Warning notes

Eventually the business server (which record your account data) will be off-line because it's currently hosted on a machine which isn't everytime online. As soon as we get enough money, we will provide a dedicated 24h server.
